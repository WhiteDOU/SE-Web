奔跑吧，少年！金锤社校园定向越野活动
发布时间：2018-05-16 10:26:11  来源：金锤社  点击：124
      一起来丰富校园生活，增强辨识地图的能力。一起来锻炼身体素质，成为更好的自己。一起来逛遍校园，更了解我们这个可爱的家。

 

         

 

活动内容：

      校园定向越野是一项智慧与体力兼备的活动，而选手能否在不同环境压力下果断地做出决策是非常重要的。参加的选手需要根据地图选择最优路线，以徒步的方式在最短时间内寻找到每一个宝藏的可能点，并在途中通过团队合作完成若干任务补全缺损地图，（提前剧透一下，将有两个任务是地球科学知识探秘问答哦！），拿到每一个地点的标记并返回终点，用时最少的队伍获得胜利。在这项活动中，不错的体力、果敢的脑力和细致的观察是最终制胜的法宝。

 

活动时间：

      2018.06.01 9：00~11：00

 

活动地点：

      东北大学南湖校区

 

活动单位：

      主办：东北大学学生社团联合会

      承办：金锤社
