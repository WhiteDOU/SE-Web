东北大学研习社
您现在的位置：首页 >社团在线 >思想政治类 >东北大学研习社

社团简介：

  东北大学研习社成立于2015年，立足于“有信仰、有担当、有思想、有内涵”的学院人才培养目标，以“青年思想家沃苗计划”为统领，遵循“立志言为本，修身行为先”的原则，以“博学善思，贯行新知，实践笃行，众志共赢”为社训，以推进学生政治理论学习、不断提升学生马克思主义思想水平为目的，以正确的舆论导向丰富校园文化生活为宗旨，为我校大学生进一步学习马克思主义理论知识提供了一个良好的平台。
