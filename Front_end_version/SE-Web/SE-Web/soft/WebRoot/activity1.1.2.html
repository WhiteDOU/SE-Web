彩绘奔跑？就是要燥起来
发布时间：2018-05-20 14:56:26  来源：心里潜能开发协会  点击：124
      以奔跑为主题，呼吁同学们在繁忙学习生活之余积极参加体育锻炼，强身健体；以彩虹为引导，呼吁大家以积极乐观的心态待人接物；以团队素拓活动为辅助，呼吁大家注重团队精神的培养，学会同团队成员精诚合作。

 

         

 

活动内容：

      预备：奔跑前，参与者领取小纸条，写上自己的名字（将纸条放入抽奖箱，作为抽奖的材料）。领取白色T恤，自己动手，在T恤上画上自己喜欢的图案，写上自己喜欢的文字。完成创作后，参与者听从现场工作人员的安排，一起奔跑。

      开始：建筑与体育馆之间>>>信息楼东北方向的马路>>>图书馆前马路北面>>>起点小游戏等着你哦~

      成功：奔跑结束后，在涂鸦墙上留下你的痕迹吧，还有抽奖环节的哦~主持人从在场参与者中随机抽选三名幸运儿获得幸运奖。

 

活动时间：

      2018.6.9下午15：00开始

 

活动地点：

      文管楼前马路出发，经过建筑西侧、信息东侧、生命东侧，回到文管前。

 

活动单位：

      主办：东北大学学生社团联合会

      承办：心理潜能开发协会
