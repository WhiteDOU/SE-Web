东大在线网络传媒工作室
您现在的位置：首页 >社团在线 >思想政治类 >东大在线网络传媒工作室

社团简介：

  “东大在线”网络传媒工作室，成立于 2008 年 3 月，隶属于东北大学党委宣传部，是四大校园学生媒体之一。多年来，东大在线辅 助校党委宣传部依托东北大学中英文主页、东北大学新闻网、东北大 学广播电视网进行网络文化建设工作；此外，东大在线还负责教育部中国大学生在线“长夜书香”频道及辽宁大学生在线联盟“时事新闻” 的共建工作，并取得了一系列优秀的成果，其负责研发和维护的各类网络文化产品在校内外引起了广泛影响，深得广大师生的赞誉。

 

所属学院：
